# Getting Started

## Package setup

<Steps>

### Setup a React app (with Convex)

Follow either the [React (Vite)](https://docs.convex.dev/quickstart/react) or [Next.js](https://docs.convex.dev/quickstart/nextjs) quickstart to get Convex set up in your React app.

### Install the library

```sh
npm i @xixixao/uploadstuff
```

### Setup your storage API

Continue with the [Server Setup](/getting-started/server-setup) guide.

### Build your UI

Choose from one of the provided ways to build your UI:

- [UploadButton](/api-reference/UploadButton)
- [UploadDropzone](/api-reference/UploadDropzone)
- [useUploadFiles](/api-reference/useUploadFiles)

</Steps>
